# Software Construction Project
## Turkish-English Dictionary
#### The goal of the project is to improve the vocabulary of users by creating an English-Turkish dictionary. It then aims to teach the users by testing them for a certain period of a time.
#### This project made with C# programming language and Microsoft SQL Database.
